## Design Standards for All 5 Pages

### Color Palette
```
Primary Blue: #1F3058 (dark medical blue)
Secondary Blue: #4A90E2 (trust blue)
Accent Green: #27AE60 (health/growth)
White: #FFFFFF
Light Gray: #F5F5F5
Dark Gray: #333333
CTA Green: #25D366 (WhatsApp)
CTA Red: #E74C3C (Emergency/call)
```

### Typography
```
Headings: Poppins Bold (24px - 48px)
Subheadings: Poppins SemiBold (18px - 24px)
Body Text: Inter Regular (14px - 16px)
Accent Text: Inter SemiBold (14px)
```

### Spacing Grid
```
Base: 8px
Sections: 80px vertical padding
Cards: 24px gaps
Mobile sections: 40px vertical padding
```

### Button Styles
```
Primary CTA: 
  - Background: #25D366 (WhatsApp green)
  - Color: White
  - Padding: 16px 32px
  - Border radius: 8px
  - Font: Bold 16px

Secondary CTA:
  - Background: #E74C3C (Red)
  - Color: White
  - Padding: 16px 32px
  - Border radius: 8px

Tertiary CTA:
  - Background: Transparent
  - Border: 2px #1F3058
  - Color: #1F3058
  - Padding: 14px 28px
```

### Responsive Breakpoints
```
Mobile: 320px - 767px
Tablet: 768px - 1023px
Desktop: 1024px+

Rules:
- Stacked layout on mobile
- 2-column on tablet
- 3-4 column on desktop
- Font sizes scale proportionally
```

### Image Guidelines
```
Doctor photo: 400x500px (portrait)
Service icons: 80x80px (SVG preferred)
Testimonial avatars: 60x60px (circular)
Hero background: 1920x1080px (or larger)
Mobile: Optimized for 375px width
Format: JPG (photos), SVG (icons)
```

## Implementation Checklist for Each Page

When designing, ensure:
- [ ] Mobile design (320px width)
- [ ] Tablet design (768px width)
- [ ] Desktop design (1024px+ width)
- [ ] All CTAs clickable/interactive
- [ ] Color contrast WCAG AA compliant
- [ ] Fonts readable at all sizes
- [ ] Whitespace balanced
- [ ] Icons consistent style
- [ ] Forms simple (2-3 fields max)
- [ ] Mobile CTA bar visible
- [ ] FAQ accordion functional
- [ ] Location map placement clear
